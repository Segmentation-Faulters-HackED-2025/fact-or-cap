Fact-or-Cap Documentation

Utilizing the Google Fact Checking API, the Fact or Cap Chrome extension serves to accurately verify any highlighted piece of information found online. It's ease of use renders it a useful companion whilst browsing the internet, as simply highlighting a chosen text prompts open the Fact or Cap Chrome extension!

1. 

        git clone https://github.com/Segmentation-Faulters-HackED-2025/fact-or-cap.git
2. Go to chrome://extensions. 3. Activate developer mode in top right 4. Upload file from "Load unpacked" in top left

Open any webpage in Chrome. Use your mouse to highlight the sentence or phrase you want to fact-check. 2. Accessing the Extension Right-click on the highlighted text. In the context menu, select "Fact-Check This" (or the extensionâ€™s name). The extension popup will open with the highlighted text automatically filled in. 3. Running the Fact-Check Click the "Check Facts" button in the extension popup. The extension will analyze the statement using reliable sources and databases. 4. Viewing the Results If the statement is accurate, the extension will confirm its validity. If the statement is misleading or false, it will display: A brief explanation of why it's incorrect. Sources that provide correct information. If the statement is uncertain, the extension will suggest further reading.
